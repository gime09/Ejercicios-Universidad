# Piedra, Papel o Tijera

Juego de consola desarrollado en Python para practicar lógica de programación y manejo de entradas por terminal.

## Descripción

Este proyecto implementa el clásico juego de "piedra, papel o tijera" entre el usuario y la computadora. El juego se desarrolla en rondas, mostrando el marcador actualizado y determinando al ganador después de alcanzar dos victorias.

## Tecnologías

- Python 3
- `colorama` para estilos en consola

## Requisitos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- Python 3.x
- `pip` para instalar dependencias

## Instalación

1. Clona el repositorio.
2. Navega hasta la carpeta del proyecto.
3. Instala la dependencia requerida:

```sh
python3 -m pip install colorama
```

## Ejecución

Desde la raíz del proyecto, ejecuta:

```sh
cd game
python3 main.py
```

## Cómo jugar

- Ingresa una de las siguientes opciones:
  - `piedra`
  - `papel`
  - `tijera`
- Escribe `salir` para finalizar la partida.

## Estructura del proyecto

- `main.py`: lógica principal del juego.
- `hello.py`: script de ejemplo simple.
