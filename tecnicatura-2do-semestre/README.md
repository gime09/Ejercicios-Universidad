## Descripción general

Este proyecto reúne una colección de ejercicios, prácticas y proyectos desarrollados durante la formación en programación. Su finalidad es consolidar conocimientos en:

- Programación orientada a objetos en Java
- Desarrollo web básico con JavaScript
- Programación general y automatización con Python
- Gestión de repositorios y flujo de trabajo con Git y GitHub

## Objetivo del repositorio

El repositorio funciona como un portafolio académico de trabajos realizados en clase, permitiendo:

- Practicar conceptos teóricos en ejercicios reales
- Mantener un registro de avances por semestre
- Reforzar la organización de proyectos y archivos
- Servir como material de referencia para repaso y estudio

## Estructura del proyecto

La organización principal del repositorio es la siguiente:

- `class_git/`: ejercicios y apuntes relacionados con Git, commits, branches y flujo de trabajo colaborativo.
- `java/`: prácticas y proyectos desarrollados en Java, incluyendo ejercicios de lógica, condicionales, ciclos, matrices y programación orientada a objetos.
- `javaScript/`: actividades y ejemplos con JavaScript, abordando fundamentos del lenguaje y ejercicios de clase.
- `python/`: ejercicios y proyectos de Python desarrollados en distintos niveles del curso.
- `readme/`: materiales complementarios y apuntes de referencia.

```text
Tecnicatura/
├── class_git/
├── java/
├── javaScript/
├── python/
└── readme/
```

## Tecnologías y lenguajes

### Java
Se utilizan ejercicios relacionados con:

- Variables y tipos de datos
- Operadores y estructuras de control
- Clases, objetos y métodos
- Condicionales, ciclos y arreglos

### JavaScript
Se trabajan fundamentos del lenguaje, incluyendo:

- Sintaxis básica
- Variables y funciones
- Estructuras de control
- Lógica de programación aplicada a ejercicios de clase

### Python
Se exploran:

- Introducción a Python
- Estructuras de control
- Ejercicios prácticos por clases
- Desarrollo de pequeñas soluciones y automatizaciones

### Git y GitHub
El repositorio también incorpora prácticas de control de versiones con Git, incluyendo:

- Inicialización de repositorios
- Manejo de commits
- Trabajo con archivos y historial
- Organización de carpetas y control de cambios

## Cómo usar este repositorio

1. Cloná el repositorio en tu máquina local.
2. Navegá por las carpetas según el lenguaje o tema que necesites estudiar.
3. Abrí cada proyecto en tu editor preferido.
4. Ejecutá los ejercicios o proyectos siguiendo la estructura de cada carpeta.

```bash
git clone https://github.com/gime09/Tecnicatura.git
cd Tecnicatura
```

## Recomendaciones para estudiar

- Revisá primero los ejercicios básicos antes de avanzar a proyectos más complejos.
- Mantén el orden por carpetas y clases para seguir el progreso académico.
- Practicá cada tema con ejemplos propios para reforzar la comprensión.
- Usá Git para registrar cambios y mantener un flujo de trabajo ordenado.
