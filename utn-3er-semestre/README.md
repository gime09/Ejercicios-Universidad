# Repositorio de Programación III

Este repositorio reúne las prácticas, ejercicios, proyectos y material de estudio desarrollados en la materia de Programación III durante dos comisiones:

- Programación III - Lunes
- Programación III - Miércoles

El contenido está organizado por lenguaje y por tema de clase, con ejemplos y ejercicios en JavaScript, Python y Java.

## Objetivo del repositorio

El propósito de este workspace es servir como carpeta de estudio y práctica para reforzar conceptos fundamentales de:

- programación orientada a objetos,
- manejo de errores y excepciones,
- estructuras de datos y lógica algorítmica,
- conexión con bases de datos,
- programación con interfaces y abstracciones,
- desarrollo web con HTML, CSS y JavaScript,
- trabajo con archivos y registros en Python.

## Estructura general

El repositorio está dividido en dos grandes bloques principales:

1. Programación III - Lunes
2. Programación III - Miércoles

Cada bloque contiene las clases y prácticas correspondientes, separadas por lenguaje y temática.

## 1. Programación III - Lunes

Esta comisión concentra una variedad de ejercicios prácticos con JavaScript y Python.

### 1.1 JavaScript

La carpeta de JavaScript aborda desde álgebra de programación hasta conceptos de programación orientada a objetos y tareas web básicas.

#### Clase 01 - Proyecto MundoPC con JavaScript
- Objetivo: practicar la modelización de componentes de una computadora mediante un sistema de ventas.
- Archivo principal: `sistema-ventas-js.js`.
- Tema central: clases, objetos, composición y lógica de negocio básica.

#### Clase 02 - Modo strict y sobreescritura
- Se trabaja con el modo estricto de JavaScript (`"use strict"`).
- Se introducen los conceptos de sobreescritura y reutilización de comportamiento en clases.
- Carpeta principal: `Leccción02/`.

#### Clase 03 - Polimorfismo y instanceof
- Ejercicios sobre herencia y la sustitución de comportamiento mediante polimorfismo.
- Uso de `instanceof` para verificar tipos dentro de jerarquías de objetos.
- Carpeta principal: `Leccion03/` y el subproyecto `polimorfismo_mundopc/`.

#### Clase 04 - Manejo de Errores
- Contiene ejercicios de control y manejo de errores en JavaScript.
- Se practica validación y prevención de fallos en ejecución.

#### Clase 05 - Función flecha (arrow function)
- Se estudia la sintaxis de funciones flecha y cómo simplifican definiciones de funciones.
- Se compara su uso con funciones tradicionales.

#### Clase 06 - Callback, setTimeout y setInterval
- Se analiza la ejecución asíncrona y el manejo de callbacks.
- Se practican timers y la programación basada en eventos.

#### Clase 07 - Prácticas con Js
- Proyecto web con HTML, CSS y JavaScript.
- Incluye una práctica sobre el problema de las N reinas.
- Archivos principales:
  - `index.html`
  - `script_n-reinas.js`
  - `style.css`

#### Clase 08 - Prácticas con JavaScript: El salto del caballo
- Ejercicio clásico del recorrido del caballo en un tablero de ajedrez.
- Tiene enfoque en lógica algorítmica y visualización.

### 1.2 Python

La carpeta de Python reúne el manejo de excepciones, lectura y escritura de archivos, trabajo con bases de datos PostgreSQL y organización de proyectos pequeños.

#### Clase 1 - Excepciones
- Ejercicios sobre `try`, `except`, `finally` y control de errores.

#### Clase 1Teoría
- Material de apoyo teórico introductorio.

#### Clase 2 - Manejo de Archivos
- Prácticas sobre lectura, escritura y manipulación de archivos.

#### Clase 2 Teoría
- Parte documental y bibliográfica relacionada con frameworks y conceptos web.

#### Clase 3 Parte 1 - Python
- Proyecto de catálogo de películas.
- Tiene una estructura organizada para manejar registros y datos de películas.

#### Clase 4 - PostgreSQL y Python
- Integración entre Python y bases de datos PostgreSQL.
- Se trabajan consultas, conexión y acceso a datos.

#### Clase 5 - Registros con PostgreSQL y Python
- Manejo de inserciones, consultas y registros en base de datos.

#### Clase 6 - Transacciones en PostgreSQL y Python
- Se estudian transacciones, confirmación y control de cambios en la BD.

#### Clase 7 - Capa de Datos, Logging y PostgreSQL Parte 1
- Introducción a la capa de datos, registro de eventos y buenas prácticas de conexión.

#### Clase 8 - Capa de Datos, Logging y PostgreSQL Parte 2
- Continuación del trabajo de acceso a datos y logging.

#### clase 7 Particulas
- Proyecto o ejercicio práctico relacionado con partículas, animaciones o lógica visual.

## 2. Programación III - Miércoles

Esta comisión está orientada principalmente al lenguaje Java y a la programación orientada a objetos con énfasis en encapsulamiento, herencia, interfaces y manejo de excepciones.

### 2.1 Java

#### Clase1 - Laboratorio en Java
- Carpeta principal: `Lección01/`.
- Proyecto principal: `MundoPc`.
- Se trabaja con clases, relaciones entre objetos y lógica de dominio.

#### Clase2 - Bloques y mucho más
- Contiene ejercicios sobre:
  - bloques de inicialización,
  - argumentos variables,
  - enumeraciones.

#### Clase3 - Teoría
- Material documental para apoyar el aprendizaje de la clase.

#### Clase3 - forEach, Autoboxing, Unboxing, Modificadores de acceso
- Se exploran:
  - recorrido de colecciones con `forEach`,
  - autoboxing y unboxing,
  - modificadores de acceso para encapsulamiento.

#### Clase4 - Sobreescritura, Polimorfismo e instanceof
- Ejercicios sobre:
  - sobrescritura de métodos,
  - polimorfismo,
  - validación de tipos con `instanceof`.

#### Clase5 - Conversión de objetos, Downcasting, Upcasting, hashCode, equals, abstracción
- Trabajo con conversiones entre tipos de objetos.
- Uso de `hashCode`, `equals` y clases abstractas.

#### Clase7 - Interfaces en Java
- Ejercicios con interfaces, abstracción y desacoplamiento.
- Importancia en el diseño de software y contratos de clases.

#### Clase8 - Excepciones en Java
- Manejo de errores en Java, con enfoque en JavaBeans y flujo controlado de fallos.

#### Clase8 - Teoría
- Documentación adicional relacionada con la clase.

## Cómo navegar este repositorio

### Para JavaScript
- Abre archivos `.js` directamente o ejecuta el proyecto en un navegador si contiene HTML/CSS.
- Los ejercicios de lógica pueden correrse en Node.js o en un navegador moderno.

### Para Python
- Ejecuta los scripts con Python 3.
- Si el proyecto usa PostgreSQL, asegúrate de tener la base de datos configurada y las librerías necesarias.

### Para Java
- Los proyectos están organizados como carpetas de práctica.
- Puedes abrirlos en un IDE como IntelliJ IDEA, Eclipse o VS Code con soporte para Java.
